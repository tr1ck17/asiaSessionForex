Asia Session USDJPY Forex Trading Application (Automated)

MUST create a .env file that contains values for:
  OANDA_API_TOKEN
  OANDA_ACCOUNT_ID
  OANDA_ENV
OANDA_ENV = practice -> this will work for paper trading.
For trading with live accounts:
OANDA_ENV = live

OANDA_API_TOKEN can be generated by going to your OANDA brokerage account and clicking "Generate" under API Code section
OANDA_ACCOUNT_ID: is the series of numbers in your Account Details section

Clone to active directory, cd into cloned repo directory, and activate virtual environment with command: python -m venv .venv
Activate venv with command: .venv\Scripts\activate (.venv\Scripts\Activate.ps1 if prior command doesn't work)
Then, command: pip install -r requirements.txt
